  Field        Rule ID         Logic
  ──────────   ─────────────   ──────────────────────────────────────────────
  NAME         name-min        value.trim().length >= 3
  EMAIL        email-fmt       /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(value)
  PASS         pass-min        value.length >= 8
  PASS         pass-upper      /[A-Z]/.test(value)
  PASS         pass-num        /[0-9]/.test(value)
  PHONE        phone-valid     /^[\+]?[\d\s\-\(\)]{7,15}$/.test(value)
  URL          url-https       /^https:\/\/.+\..+/.test(value)

  Password Strength Scoring (0–6 points):
    +1   length >= 8
    +1   has uppercase letter
    +1   has digit
    +1   has lowercase letter
    +1   has special character
    +1   length >= 12 (bonus)

  Score → Level mapping:
    0–2   WEAK    → bar color: --error  (#ff6584)
    3–4   MEDIUM  → bar color: --warn   (#ffd166)
    5–6   STRONG  → bar color: --valid  (#43e97b)

  Active bars = ceil(score / 1.5) capped at 4
