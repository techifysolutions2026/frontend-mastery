# AI Usage

This document outlines how AI was used during the development of this project.

---

## Design

The overall design of the website — layout, color scheme, typography, and visual aesthetic — was conceptualized with the help of **Claude (Anthropic)**. The AI helped shape the industrial retro-futuristic direction of the UI.

---

## HTML & CSS

Written entirely by me. The structure and styling of the interface were hand-coded without AI assistance.

---

## JavaScript

Most of the JavaScript logic was written by me. The only exception is the **animated number count-up function**, where AI was used to generate the implementation. All other interactivity, toggle logic, and clock functionality was written by me.

---

## Summary

| Area | Author |
|---|---|
| Design | Claude (AI-assisted) |
| HTML | Me |
| CSS | Me |
| JS — Animated counter | Claude (AI-assisted) |
| JS — Everything else | Me |
